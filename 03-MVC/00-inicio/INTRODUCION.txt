¿Que es y como funciona?
Es un patron de diseño de software que se encarga de separar
la logica de negocio, de la interfaz del usuario, 
Separa lo que es la funcionalidad de lo que se muestra
al usuario.
El MVC divide las aplicaciones en 3 niveles
1. Modelo -> El modelo es la logica de negocio, es la parte
    de la clase y los metodos que se comunican con la base de datos
2. Vista -> Es la encargada de mostrarse la informacion
    al usuario de forma grafica
3. Controlador -> Es el intermediario entre la vista y el
    modelo y se encarga de controlar las interacciones que hace
    el usuario con la vista.
